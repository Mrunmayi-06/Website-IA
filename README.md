# 🌱 InAmigos Foundation Website

A modern, responsive, and impact-driven NGO website built for **InAmigos Foundation** — a youth-led nonprofit organization working across food security, education, environmental sustainability, women empowerment, rural development, and animal welfare.

---

## ✨ About the Project

This website was designed to represent the mission and impact of **InAmigos Foundation** in a clean, engaging, and professional way.

The platform highlights:

- NGO mission & vision
- Active social projects
- Donation campaigns
- Volunteer opportunities
- Impact statistics
- Testimonials
- Contact & collaboration forms
